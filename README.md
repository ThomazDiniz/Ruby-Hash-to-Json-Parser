# Ruby Hash to Json Parser

Ferramenta web para converter entre Ruby Hash e JSON. Permite conversão bidirecional em tempo real com formatação automática.

## Acesso

Acesse a aplicação em: https://thomazdiniz.github.io/Ruby-Hash-to-Json-Parser/

## Como usar

1. Cole um Ruby Hash no painel esquerdo para converter para JSON
2. Cole um JSON no painel direito para converter para Ruby Hash
3. Use o botão "Beautify" para formatar o código
4. A conversão acontece automaticamente enquanto você digita
